# WealthBuilder (Formerly TeenTycoon)

**A Premium Fintech Education & Planning Platform**

WealthBuilder is a modern, data-driven web application designed to help ambitious individuals learn how money works, map out their financial future, and make calculated wealth-building decisions. 

We completely redesigned the platform from a "gamified" MVP into a sleek, professional financial framework. It is engineered with the credibility, aesthetics, and utility of top-tier platforms like Bloomberg and modern fintech startups.

## 🚀 Key Features Built

### 1. Interactive Tool Suite (10 Custom Calculators)
We built an entire dynamic Modal system housing 10 bespoke financial tools, turning abstract concepts into actionable insights:
- **Net Worth Simulator:** Projects 10-year wealth trajectories based on current savings and monthly contributions.
- **Compound Interest Visualizer:** A dynamic visualizer demonstrating the power of exponential growth on initial investments.
- **How Rich Are You?:** Computes your net worth percentile ranking compared to average Americans.
- **Side Hustle ROI Calculator:** Calculates Break-Even points and "True Hourly Wages" factoring in startup costs and time investment.
- **Real-Time Portfolio Builder:** A live visualization of an "Aggressive Growth" asset allocation (S&P 500, International, Bonds).
- **Financial Decision Simulator:** A "Rent vs Buy" calculator that models the *opportunity cost* of locking up down payments vs investing in index funds.
- **Finance Career Salary Map:** Data-driven expected compensation mapping (Base + Bonus) for heavy-hitting roles like IB Analysts and PE Associates.
- **Investor Portfolios Deconstructed:** A breakdown of Warren Buffett's highest-conviction value investing holdings (AAPL, BAC, AXP).
- **Market Events Explainer:** Expandable, easy-to-digest definitions of critical macroeconomic forces (Inflation, Interest Rates, Bear Markets).
- **Finance Myths Debunked:** Exposing mainstream fallacies (e.g., "Renting is throwing money away") with mathematical reality checks.

### 2. GenAI Finance Mentor
An integrated, beautifully styled AI chat assistant powered by the Purdue Model GenAI API. Users can access the Mentor directly from the main navigation to get instant, digestible explanations of complex financial jargon or investing strategies in real-time.

### 3. Yahoo Finance Market Connections
A "Featured Insights" and live market ticker strip engineered to connect with Yahoo Finance APIs, keeping users grounded in real-world data and trusted financial analysis.

### 4. Premium Aesthetic Redesign
- Transitions from a neon/gamer theme to a high-end, credible financial identity.
- Single-page, smooth-scrolling architecture with glassmorphism `(glass-card)` UI elements.
- Dedicated learning modules including the **$0 to Wealth Roadmap** and **"What $100 Invested in 1990 is Worth Today"** visualizations.

## 💻 Tech Stack
- **Frontend Core:** Pure HTML5, CSS3 (Vanilla), and JavaScript.
- **Architecture:** Single Page Application (SPA) architecture utilizing custom semantic routing (`app.navigateTo()`).
- **Icons & Assets:** FontAwesome 6 integration.
- **Integrations:** Designed for Purdue Model GenAI and YahooFinance integrations.

## 🛠 Deployment
This codebase is fully static and browser-ready. It can be hosted entirely through GitHub Pages, Vercel, or simply opened locally by launching `index.html`.
